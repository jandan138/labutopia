为什么它在本项目里重要

LabUtopia collection 使用 python main.py --config-name ...；training 使用 python train.py --config-name=...；Isaac Sim 环境安装按 README 使用 isaacsim[all,extscache]==5.1.0。EBench assets/dataset 下载使用 huggingface-cli，评测使用 gmp submit、gmp eval、gmp status。

从 RL 看 把概念落到 trajectory

这里的关键不是把术语背下来，而是看它怎样改变 episode、observation、action、success 和 dataset distribution。

从代码和资产落地

命令不是孤立记忆。每条命令背后都有 contract：config-name 决定 task；dataset_path 决定训练数据；gmp eval 依赖 server、run_id、worker_ids、lift2 contract。

Command sheet
python main.py --config-name level1_pick
python main.py --config-name level5_Navigation --headless --no-video
python train.py --config-name=train_act_image_workspace
huggingface-cli download InternRobotics/EBench-Assets --local-dir saved --repo-type dataset
gmp submit ebench/generalist/val_train --run_id my_run
gmp eval -a r5a -g lift2 --worker_ids 0

读论文和跑实验时怎么用

执行前先确认当前工作目录、Python env、Isaac Sim version 和 GPU 是否满足要求。

本节要记住
  • 命令要和配置一起读。
  • 跨项目命令不要混用。
  • EBench 命令不能直接驱动 LabUtopia main.py。
