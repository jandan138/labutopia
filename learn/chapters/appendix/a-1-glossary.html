Glossary：中英文术语对照

本附录把书中反复出现的 English terms 放在一起，便于你读论文、代码和 docs 时保持同一套词表。

为什么它在本项目里重要

SimulationApp 是 Isaac Sim 应用入口；World 是 simulation wrapper；Stage 是 OpenUSD scene graph；prim path 是 USD 对象路径；articulation 是机器人关节系统；RMPFlow 是 Franka 运动生成/控制工具；BaseTask 负责 state；BaseController 负责 action；DataCollector 写 episode；LeRobot 是 EBench-Dataset 使用的数据格式生态。

从 RL 看 把概念落到 trajectory

这里的关键不是把术语背下来，而是看它怎样改变 episode、observation、action、success 和 dataset distribution。
