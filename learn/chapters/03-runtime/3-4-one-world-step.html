为什么它在本项目里重要

如果 world stopped，controller 需要 reset；如果 world playing，程序先检查 controller 或 task 是否需要 reset。reset 分支会释放 video writer、重置 controller、判断 episode 数是否达到 cfg.max_episodes、再重置 task。非 reset 分支才调用 task.step() 取 state。

从代码和资产落地

拿到 state 后，controller 返回 action, done, is_success。如果 action 存在，articulation controller 应用动作；如果 done，打印 failure reason 并调用 task.on_task_complete(is_success)。最后，若开启视频，程序把 camera_display 拼成一张 OpenCV 画面并写入 mp4。
